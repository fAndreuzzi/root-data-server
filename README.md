# (ROOT) data server

RESTful web service for storage and metadata extraction of Open Data repositories.

## Technologies

### Software stack

- Java 11
- Quarkus
    - RESTEasy Reactive
    - Jackson
- Jakarta EE CDI
- MongoDB Java driver
- Apache Commons CSV
- Apache Commons Lang3
- Slf4j + Logback

### Build tools

- Quarkus CLI
- Gradle (Kotlin DSL)
